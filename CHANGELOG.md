# Changelog

## 1.0.0

Initial public release.

- Zabbix 7.4 JSON import template
- Digest authentication support
- Host macros: `{$SHELLY_HOST}`, `{$SHELLY_USER}`, `{$SHELLY_PASS}`
- Switch mode monitoring for `switch:0` and `switch:1`
- Cover mode monitoring for `cover:0`
- Input monitoring for `input:0` and `input:1`
- Optional cover position handling without noisy preprocessing errors
